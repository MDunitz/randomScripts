# Data Sources and References

## Critical Minerals Data

All data points in this project are sourced from authoritative government and scientific publications.

### Primary Data Sources

1. **USGS Mineral Resources Data System (USMIN)**
   - URL: https://mrdata.usgs.gov/
   - Comprehensive database of mineral occurrences and deposits in the United States
   - Updated regularly with new discoveries and resource assessments

2. **USGS Mineral Resources Program**
   - URL: https://www.usgs.gov/programs/mineral-resources-program
   - Critical Minerals List and resource assessments

3. **EPA National Priorities List (NPL) - Superfund Sites**
   - URL: https://www.epa.gov/superfund/national-priorities-list-npl-sites
   - Contamination data for mining and smelting legacy sites
